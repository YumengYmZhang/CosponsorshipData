The Archive.zip data contains 93th to 118th congressional data in .RData format. 
Each congress has two lists (H: House and S: Senate), and each list has three items: 
1. Cosponsorship presented via matrix.
    - Rows are legislator; columns are billID; 2 means sponsor; 1 means cosponsor; 0 means no response.
3. Legislator's information including ID, name, party affiliation and state.
4. Bill's information including ID, introduced date, title, area, status, sponsor's party, number of cosponsors in Democrat, Republican and Individual.

For 93118.zip, tenure information is included.
